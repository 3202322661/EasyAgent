# 🧠 个性化记忆系统

本目录存储用户的个性化记忆数据，用于智能体随使用不断调整行为偏好。

## 文件说明

| 文件 | 用途 |
|------|------|
| `user_profile.md` | 用户基本档案（姓名、身份、背景等） |
| `preferences.md` | 偏好设置（语言风格、输出格式、工作习惯等） |
| `interaction_history.md` | 交互历史摘要（已完成的重大任务、常用功能等） |
| `knowledge_base.md` | 用户关注的知识点和领域兴趣 |

## 记忆更新机制

- 每次对话结束时自动提取关键偏好信息并更新对应文件
- 采用增量写入策略，保留历史 vs 覆盖旧偏好
